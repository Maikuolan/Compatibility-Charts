( ✔️ Current versions – ✔️ Outdated/Future versions ) + Compatible/Available.
( ✔️ Current versions – ✔️ Outdated/Future versions ) + Not tested, but expected to be compatible/available.
( ❓ Current versions – ❓ Outdated/Future versions ) + Not tested and unknown compatibility.
( ✔️ Compatible – ❌ Not compatible – [anything red-on-red] ) + Contains critical flaws/vulnerabilities, and should be avoided (PHP CVEs – PHP releases).
➖ Not applicable/relevant.
➖ Not supported.
✱ Current versions.
✱ Outdated versions (actively supported).
✱ Outdated versions (security support only).
✱ Outdated versions (end of life [EoL]; if you use these versions, please update immediately).
✱ Future versions (not yet released; currently in development).

Only relevant versions will be listed here. Irrelevant information is to be avoided and versions should be delisted upon gaining irrelevancy. Generally, versions should be considered "relevant" if (1) they're currently still being actively used by a significant portion of the userbase of the package in question, (2) are the the latest currently available versions, (3) have been announced as being planned to be released within the near future (in the case of future releases), (4) have been released relatively recently, or (5) are still currently actively being supported. However, discretion should be used as so that this determination is made on the basis of the needs of users and on the basis of whether it's useful; If it's useful, it should be included, and if it's not useful, it shouldn't be included.

If you find any errors, would like to add to the list or make some changes, please send a pull request to the GitHub repository for this page.
