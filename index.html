[L] Latest available/tagged version on this branch.
[NS] Version isn't normally supported anymore. (This doesn't indicate abandonment nor EOL. It indicates that when seeking support for this version, the most likely outcome will be a recommendation to update to a newer version; i.e., there may be bugs present within this version, fixed since newer versions, with updating being a much more efficient and practical solution than attempting to provide support for this version).
[TBA] Version hasn't been released yet. (Planned or still under development).
[CR] Contextual release. (Mightn't have the same availability as other releases, or may be available only in specific contexts).

[A] This version/release is an alpha. You could use it in production, but doing so is not recommended.
[B] Critical bugs or unintended breaks detected in this version (meaning that the version is effectively unusable). Avoid usage!
[BC] This version/release significantly breaks backwards-compatibility. When updating from an earlier version, review all changes carefully.
[F] Final expected version. Future releases aren't expected on any branch. This generally suggests abandonment or EOL ("end of life") status.
[FB] Final expected version on this branch. Future releases aren't expected on this branch, but may occur on other, newer braches.
[V] Security vulnerabilities have been identified belonging to this version/release. Avoid usage!
